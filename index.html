Flawless-Semantics

Based on the well known semantic.gs by Tyler Tate, Flawless-Semantics shares many of the same goals that semantic.gs such as allowing:

  1. column, gutter widths and the number of columns to be set;
  2. switching between pixel and percentage based layouts;
  3. responsive fixed alayouts, all without any unsemantic .grid_x classes in your markup.

As is evident from the examples below, unlike semantic.gs this grid uses a column-gutter-column approach, I have provided as many examples as I can on this page and the original semantic.gs examples can be accessed via the links below. There is no need for a omega classes as each row is created exactly one gutter width larger.

To find out more about Flawless-Semantics please see the readme located here
